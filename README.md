# Arduino-led-project
This project demonstrates how to control multiple LEDs using an Arduino Uno.
## DESCRIPTION:
This project demonstrates how to control multiple LEDs using an Arduino Uno. Five LEDs are connected to the Arduino and programmed to glow in different patterns. The LEDs turn ON and OFF sequentially with a time delay, helping beginners understand digital output control and Arduino programming.
## COMPONENTS REQUIRED:
1. Arduino Uno
2. Breadboard
3. 5 LEDs
4. 5 × 220Ω Resistors
5. Jumper Wires
6. USB Cable
## OBJECTIVE:
- Learn to interface multiple LEDs with Arduino Uno.
- Understand digital output using Arduino.
- Generate different LED glow patterns.
- Gain hands-on experience with basic electronics and programming
